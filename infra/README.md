# NovaMart Infrastructure

Infrastructure-as-code and environment configuration for NovaMart.

- AWS:
  - API Gateway, Cognito, Lambda, RDS (PostgreSQL), DynamoDB, S3, CloudWatch, IAM
- Kubernetes (EKS):
  - Cluster configuration, namespaces, service accounts, and ingress
- Service mesh:
  - Istio configuration for mTLS, traffic policies, and observability
